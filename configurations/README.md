# Configurations.

This directory contains all my tool configurations.

- Each tool gets a separate directory for its respective configurations.
- Each tool's configuration directory gets `stow`-ed to the `$HOME` directory.
- The inner directory/file structure of tool configuration directories' must follow their respective paths from `$HOME`

# Karabiner Configuration

I'm using [goku](https://github.com/yqrashawn/GokuRakuJoudo) to auto-generate the
  [karabiner.json](./.config/karabiner/karabiner.json) file. So, if you want to referency my
  configuration, you should look at the `goku` configuration file
  [here](./.config/karabiner/karabiner.edn).
